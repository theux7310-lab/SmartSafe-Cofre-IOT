# SmartSafe Cofre Eletrônico Inteligente

## Projeto de Internet das Coisas (IoT)

O SmartSafe é um sistema de segurança desenvolvido utilizando Arduino Uno, teclado matricial 4x4, servo motor, LEDs e buzzer.

## Componentes

- Arduino Uno R3
- Teclado Matricial 4x4
- Servo Motor SG90
- Buzzer Ativo 5V
- LED Verde
- LED Vermelho
- Resistores 220Ω
- Protoboard
- Jumpers

## Funcionamento

1. O usuário digita a senha.
2. O Arduino verifica a senha.
3. Se correta:
   - LED verde acende
   - Buzzer confirma
   - Servo abre a trava
4. Se incorreta:
   - LED vermelho permanece ativo
   - Buzzer sinaliza erro
